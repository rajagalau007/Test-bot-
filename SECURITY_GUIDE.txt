RULE #1: NEVER SHARE YOUR PRIVATE KEY
======================================

Your private key (or seed phrase) is like the password to your bank account.
ANYONE with your private key can steal ALL your crypto and NFTs.

NEVER:
‚ùå Share your private key with anyone
‚ùå Type it into any website
‚ùå Send it via email, Telegram, Discord, etc.
‚ùå Store it in a file on your computer
‚ùå Take a screenshot of it
‚ùå Share it with "support" (scammers impersonate support)
‚ùå Store it in a bot or automated system

LEGITIMATE WEBSITES NEVER ASK FOR YOUR PRIVATE KEY!


RULE #2: USE A BURNER WALLET FOR NEW MINTS
===========================================

A "burner wallet" is a separate wallet with limited funds that you use
for new, unverified NFT projects.

Why?
- If the project is a scam, you only lose what's in that wallet
- Malicious contracts can't drain your main wallet
- Limits exposure to new/unknown smart contracts

How to create a burner wallet:
1. Create a new MetaMask account (or other wallet)
2. Only transfer what you're willing to lose
3. Use this wallet ONLY for new NFT mints
4. Transfer valuable NFTs to your main wallet later

RULE #5: CHECK THE MINT WEBSITE URL
====================================

Scammers create fake websites that look identical to real ones.

ALWAYS verify:
‚ñ° URL matches official Twitter/Discord announcements
‚ñ° HTTPS (padlock icon in browser)
‚ñ° No typos in domain name (opensae.io vs opensea.io)
‚ñ° Bookmark the official site to avoid phishing

Common scam tactics:
- Homoglyph attacks (using similar-looking characters)
- Extra characters (opensea-nft.io instead of opensea.io)
- Different TLD (.com vs .io vs .xyz)
- Subdomain tricks (opensea.scammer.com)

ALWAYS cross-reference the URL with official sources!


RULE #6: UNDERSTAND GAS FEES
=============================

Minting an NFT requires paying "gas fees" to the blockchain.

Before confirming:
‚ñ° Check current gas prices (use Etherscan Gas Tracker)
‚ñ° Understand you'll pay gas even if the NFT is "free"
‚ñ° Gas fees can range from $5 to $500+ depending on network
‚ñ° Failed transactions still cost gas
‚ñ° You can adjust gas to save money (but may fail)

Tip: Mint during off-peak hours (nights/weekends) for lower gas.


RULE #7: WALLET SECURITY BASICS
================================

Hardware Wallets (Recommended):
- Ledger or Trezor
- Private keys never leave the device
- Most secure option for valuable holdings

Software Wallets:
- MetaMask, Phantom, etc.
- Convenient but less secure
- Good for burner wallets

Security tips:
‚ñ° Use a strong password
‚ñ° Enable 2FA where available
‚ñ° Keep software updated
‚ñ° Use a password manager
‚ñ° Never use public WiFi for transactions
‚ñ° Don't install unknown browser extensions
‚ñ° Clear cache/cookies regularly


RULE #8: RECOGNIZE COMMON SCAMS
================================

Fake Airdrops:
- "You won a free NFT! Click here!"
- Links to malicious contracts that drain wallets

Phishing:
- Fake emails from "OpenSea" or "MetaMask"
- Links to fake websites that steal credentials

Rug Pulls:
- Team mints, takes money, and disappears
- Website goes offline after mint

Pump and Dump:
- Artificial hype to inflate prices
- Team sells, price crashes

Discord/Telegram Scams:
- Fake support agents (always scammers)
- Fake "mint now" announcements
- Impersonation of team members

WHAT TO DO IF SCAMMED
======================

If you think you've been scammed:

Immediate actions:
1. STOP interacting with the contract
2. Revoke all approvals (Revoke.cash)
3. Transfer remaining assets to new wallet
4. Document everything (screenshots, transaction hashes)
5. Report to platform (OpenSea, Magic Eden, etc.)
6. Report to local authorities if significant loss

Prevention for next time:
- Review what went wrong
- Learn from the experience
- Implement stricter security measures
- Don't chase losses with risky mints
