ğŸ’¡ TIPS TERMUX
===============

1. PREVENT SLEEP:
   termux-wake-lock
   (Bot tetap jalan walau screen off)

2. BACKGROUND MODE:
   python telegram_control.py &
   (& = background)

3. CHECK RUNNING:
   ps aux | grep python
   (Lihat process running)

4. VIEW LOGS:
   tail -f bot.log
   (Lihat log real-time)

5. STOP BACKGROUND:
   pkill -f telegram_control
   (Stop semua bot)

6. AUTO-START:
   # Buat script startup
   echo "cd ~/crypto-bot && python telegram_control.py &" > ~/start_bot.sh
   chmod +x ~/start_bot.sh
   # Jalankan: ./start_bot.sh
